# Polaris Components

A collection of components for Shopify app developers, based on the Polaris UI library & design system

## Documentation

Components are stored under `/components/library`

```
components
└── library
    ├── ActionCard
        ├── preview

        │   ├── Example.jsx
        │   └── index.ts
        └── tabs
            ├── example
            └── ui
        └── index.ts
```
